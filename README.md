# es6-scss-webpack-seed
Simple seed for es6 application with scss based on webpack.

## Usage:
```ssh
npm install
npm start
```

## HTML partials including:
```html
<header>
    <%= require('html-loader!./partial/header.html') %>
</header>
```

## Assets support
```html
<img src="assets/placeholder.gif" alt="asset" class="sample-img"/>

```
## Browser access:
```
http://localhost:4200/
```
